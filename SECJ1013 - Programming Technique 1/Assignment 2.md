# Assignment 2

### Assignment Instruction
This assignment must be done in pairs consisting of 2 members. There are two sets of questions and we need to follow the set number as our previous Assignment 1. The program must follow the input and output as required in the text and shown in the examples. We are required to test the programs with (but not limited to) all the input given in the examples.

### Content of Set 1
* Write a complete C++ program to implement the BMR calculator
* Apply branching/selection (if…else) to handle input validation and user choices
* Apply loop/repetition (repeat…until or do…while) for re-entering data or repeating calculations
* Define at least one user-defined function besides main() with appropriate arguments
* Program must produce output as shown in the sample, including BMR calculation and daily calorie needs
* Include option for the user to enter other data and handle Y/N responses
